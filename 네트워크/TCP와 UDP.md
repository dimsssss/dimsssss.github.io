## 비교
|  | 연결성 | 제어 | 신뢰성 | 속도 |
| ---- | ---- | ---- | ---- | ---- |
| TCP | O(3-handshake, 4-handshake) | O(혼잡 제어, 흐름 제어) | 높음(데이터의 순서를 보장, 데이터의 완전성 보장) | 느림 |
| UDP | X | X | 낮음(데이터 순서 신경쓰지 않음, 데이터가 유실되어도 신경쓰지 않음) | 빠름 |
## 연결성
TCP와 UDP 모두 socket을 이용하여 통신을 시작한다. TCP는 handshake 개념을 이용하여 연결을 확인하고 종료한다.
