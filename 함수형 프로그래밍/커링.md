## What is Currying

함수를 파라미터를 기준으로 분류할 수 있다. 예를 들어 파라미터가 하나인 경우를 단항 함수라고 한다. 두 개 인경우를 이항 함수라고 한다. 마지막으로 인자가 변하는 함수를 가변 인자 함수라고 한다.

자바스크립트에서는 이 때 가변 인자 함수를 여러 개의 단항 함수로 중첩시켜셔 표현하는 것을 커링이라고 한다.

```javascript
const sum = (a, b, c, d, e) => a + b + c + d + e 

const manualCurrySum = a => b => c => d => e => a + b + c + d + e
```
